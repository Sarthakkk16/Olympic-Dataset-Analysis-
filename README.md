# Olympic-Dataset-Analysis-
This project focuses on Exploratory Data Analysis (EDA) of the Olympic Athletes dataset to uncover trends in medal distribution, gender participation, physical attributes, and sport-wise performance across years.
| **Column Name** | **Description**                                                  |
| --------------- | ---------------------------------------------------------------- |
| **ID**          | Unique identifier for each athlete                               |
| **Name**        | Athlete’s full name                                              |
| **Sex**         | Gender of the athlete (`M` or `F`)                               |
| **Age**         | Age of the athlete during the event                              |
| **Height**      | Height (in centimeters)                                          |
| **Weight**      | Weight (in kilograms)                                            |
| **Team**        | Name of the athlete’s team or country                            |
| **NOC**         | National Olympic Committee 3-letter code                         |
| **Games**       | Edition of the Olympic Games (e.g., 2016 Summer)                 |
| **Year**        | Year the Olympics took place                                     |
| **Season**      | Season of the Olympics (`Summer` or `Winter`)                    |
| **City**        | Host city                                                        |
| **Sport**       | Sport category (e.g., Athletics, Swimming)                       |
| **Event**       | Specific event (e.g., 100m sprint)                               |
| **Medal**       | Type of medal won (`Gold`, `Silver`, `Bronze`, or `NaN` if none) |

## Key Insights

Growth of female participation over time

Countries dominating specific sports

Trends in athlete body metrics by sport

Historical medal distributions
